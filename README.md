Come richiesto vengono consegnati i seguenti: 

* Il file "report_gruppo_23.pdf";
* La cartella "report" contenente il file .tex, cioè il codice LaTeX del progetto;
* La cartella "figs" contenente le immagini;
* La cartella "code" contenente gli script "controllo_scaldatore_elettrico.m" e "animazioneNonLineare.m" e i file .slx e .slxc relativi all'implementazione in Simulink.


L'ordine di esecuzione dei file è il seguente:

1. controllo_scaldatore_elettrico.m -> Contiene, divisi per blocchi, i punti 1,2,3 e 4 del progetto.

2. simulatore_non_lineare.slx -> Contiene l'implementazione in Simulink del sistema e in particolare lo svolgimento del punto 5. Le relative considerazioni sono riportate nel pdf.

3. animazioneNonLineare.m -> Contiene l'implementazione in Matlab del punto opzionale relativo allo sviluppo di un'interfaccia grafica che mostri la dinamica del sistema.
